# Public Central School Website

This repository is a static website for Public Central School in Letang, Nepal. The project includes an informative homepage, course details, school history, and contact information.

## What was improved

- Clean, responsive page layouts
- Shared navigation and mobile menu behavior
- Smooth scrolling and accessible keyboard focus states
- Better semantic structure across pages
- Updated content sections for courses, about, and contact details

## Files

- `index.html` — Homepage with hero section and overview
- `About-us.html` — About page with mission, vision, programs, and location
- `course.html` — Course offerings with student-friendly cards
- `contact.html` — Contact page with enquiry form and map
- `style.css` — Shared responsive styling
- `main.js` — Mobile menu and contact form interaction

## Local preview

Open the HTML files directly in your browser or use a simple HTTP server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
